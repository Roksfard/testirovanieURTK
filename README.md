# testirovanieURTK
Разработать консольное приложение "Калькулятор" на C#, выполняющие основные арифметические операции. Калькулятор должен уметь складывать, вычитать, умножать, делить и возведение в степень. Пользователь должен вводить переменные с клавиатуры, предворительно выбрав нужную ему операцию. 
Интерфейс должен быть чистый и понятный. Иметь подсказки для пользователя, обработку ошибок ввода. 
Алгоритм работы:
1. Отображение доступных операции.
2. Запрос выбора операции.
3. Проверка корректного вода.
4. Запрос первой переменной.
5. Проверка корректного вода.
6. Запрос второй переменной.
7. Проверка корректного вода.
8. Выполнение вычисления.
9. Вывод результата.
10. Предложение продолжить вычисления. 
Тесткейс:
1. Выбор операции: +
2. Ввести первую переменную: 5.
3. Ввести вторую переменную: 5.
4. Нажать на Enter.
5. Вывод результата: 10.

1. Выбор операции: -
2. Ввести первую переменную: 10.
3. Ввести вторую переменную: 5.
4. Нажать на Enter.
5. Вывод результата: 5.

1. Выбор операции: *
2. Ввести первую переменную: 5.
3. Ввести вторую переменную: 5.
4. Нажать на Enter.
5. Вывод результата: 25.

1. Выбор операции: :
2. Ввести первую переменную: 5.
3. Ввести вторую переменную: 5.
4. Нажать на Enter.
5. Вывод результата: 1.

1. Выбор операции: ^
2. Ввести первую переменную: 5.
3. Ввести вторую переменную: 2.
4. Нажать на Enter.
5. Вывод результата: 25.
